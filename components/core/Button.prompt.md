Zyvora's primary call-to-action control — use for the main action on any screen (submit, upgrade, continue).

```jsx
<Button variant="primary" size="md">Get started</Button>
<Button variant="secondary" icon={<ChevronIcon/>}>Filter</Button>
```

Variants: `primary` (solid purple, main CTA), `accent` (solid clay, secondary emphasis e.g. "Upgrade"), `secondary` (outlined, default choice for non-primary actions), `ghost` (no border, toolbar/inline actions), `danger` (destructive confirm). Sizes: `sm` `md` `lg`. `disabled` dims to 45% opacity.
